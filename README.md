#Data Pipepline Automation

## About Code
This script automates the process of identifying the most recent file from the Downloads folder, moving it to the Raw_Data folder, performing data cleaning and transformation, and saving the final processed file into the Processed_Data folder.

---
## Workflow
1. Identify the latest file in the Downloads directory  
2. Move the file to the Raw_Data folder  
3. Perform data cleaning and transformation  
4. Save the cleaned file in the Processed_Data folder

## Technologies Used
- Python (Data Processing & Automation)

## Objective
- Automate repetitive manual tasks  
- Improve data accuracy and consistency  
- Reduce processing time  
- Create a structured data flow

##  Folder Structure
- Downloads → Source files  
- Raw_Data → Initial files for processing  
- Processed_Data → Cleaned and final output

## Outcome
- Eliminated manual file handling  
- Improved efficiency and reliability  
- Ensured consistent data processing workflow
